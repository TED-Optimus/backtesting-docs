visualizer documentation

  • After users generated transaction file from backtesting, they can visualize each transaction entry and exit in Candlestick chart with this library.

HOW TO USE

Example for visualizer function

  1import numpy as np
  2import pandas as pd
  3
  4from optimus.backtesting.data import Universe, UniverseSelection
  5from optimus.backtesting.indicator import Indicator
  6from optimus.backtesting.performance import Performance
  7from optimus.backtesting.portfolio import Portfolio
  8from optimus.backtesting.selection import EquityModel, PositionSizing, Reason
  9
 10# Step 1. Import visualizer
 11from optimus.backtesting.visualizer import Visualizer
 12
 13if __name__ == "__main__":
 14    # Desired output option
 15    desired_width = 1000
 16    pd.set_option("display.width", desired_width)
 17    pd.set_option("display.max_columns", 15)
 18
 19    # Step 2. Initialize stock price data with desired indicator(s).
 20    indicator = Indicator()
 21    indicator.sma([20, 50])
 22    indicator.rsi([2, 5])
 23    indicator.roc([100])
 24    indicator.average_daily_turnover([50])
 25    indicator.candle_bearish_engulfing()
 26
 27    # Initializing universe
 28    universe = Universe(selection=UniverseSelection.STOCKS, indicator=indicator)
 29    df = universe.df
 30
 31    """
 32    Entry Rules:
 33    1) RSI(2) < 5
 34    2) Close > SMA(50)
 35
 36    Exit Rules:
 37    1) RSI(5) > 40
 38    2) Exit if entered for more than 10 days
 39
 40    Ranking Criteria:
 41    Rank stocks based on Rate of change(100)
 42    """
 43
 44    # Step 3. [Optional] Create custom indicator(s), eg: df["bearish_pinbar"].
 45    threshold = df["low"] + 1 / 3 * (df["high"] - df["low"])
 46    df["bearish_pinbar"] = np.where(
 47        (df["close"] < threshold) & (df["open"] < threshold), True, False
 48    )
 49
 50    # Entry signal
 51    df["buy_signal"] = np.where(
 52        (df["average_daily_turnover_50"].shift(1) >= 2500000)
 53        & (df["close"].shift(1) > 0.3)
 54        & (df["close"].shift(1) > df["SMA_50"].shift(1))
 55        & (df["RSI_2"].shift(1) < 5)
 56        & (df["ticker"] == df["ticker"].shift(1)),
 57        df["ROC_100"].shift(1),
 58        0,
 59    )
 60    df["buy_signal"] = df["buy_signal"].fillna(0)
 61
 62    # Exit signal
 63    df["sell_signal"] = np.where(
 64        (df["RSI_5"].shift(1) > 40)
 65        & (df["ticker"] == df["ticker"].shift(1)),
 66        1,
 67        0
 68    )
 69
 70    # Time stop
 71    df["buy_date"] = df.index
 72    df["sell_signal_2"] = np.where(
 73        (df["buy_signal"].shift(10) != 0) & (df["ticker"] == df["ticker"].shift(10)),
 74        df["buy_date"].shift(10).dt.date,
 75        pd.to_datetime("2000-01-01"),
 76    )
 77
 78    # Step 4. Backup price dataframe before resetting index to non-datetime type.
 79    df_drawing = df.copy()
 80
 81    # Initializing portfolio
 82    portfolio = Portfolio(
 83        df,
 84        EquityModel.TOTAL_EQUITY,
 85        PositionSizing.EQUAL_UNIT,
 86        initial_capital=int(1e5),
 87        size=10
 88    )
 89
 90    # Step 5. [Optional] Backtest your strategy and save the transaction data into a csv file. No need to re-run if using same transaction from previous run.
 91    # Backtesting starts here!
 92    for index, frame in df.groupby("date"):
 93        days_count = pd.Timedelta(days = frame.index[0])
 94        frame.set_index("ticker", inplace=True)
 95
 96        for ticker, info in portfolio.holding():
 97            if frame.loc[ticker, "sell_signal_2"] == info["open_date"].date():  # 10 days sell signal
 98                portfolio.sell(
 99                    ticker, Reason.EXIT, index, frame.loc[ticker, "open"], days_count
100                )
101            elif frame.loc[ticker, "sell_signal"] == 1:
102                portfolio.sell(
103                    ticker, Reason.EXIT, index, frame.loc[ticker, "open"], days_count
104                )
105
106        yesterday_frame = frame.copy()
107
108        # Sort remaining stocks that have buy signals by momentum (High to low)
109        frame = frame.loc[
110            (frame["buy_signal"].values != 0) & (~frame.index.isin(portfolio.list()))
111        ].sort_values("buy_signal", ascending=False)[: portfolio.free()]
112
113        # Retrieve portfolio allocation
114        for item in frame.index.to_list():
115            portfolio.buy(
116                item,
117                index,
118                frame.loc[item, "open"],
119                days_count
120            )
121
122
123    # Measuring performance
124    performance = Performance(portfolio, show=False)
125    # Step 5b. Save the transaction from backtesting
126    performance.transactions_to_csv("transaction.csv")
127    performance.objective_function(need_print=False)
128    performance.performance_metric(need_print=False)
129
130    # Visualization for transaction
131    # Step 6. Initialize visualizer with dataframe and transaction file.
132    visualizer = Visualizer(df_drawing, path_to_transaction_file="transaction.csv")
133
134    # Step 7. Add indicators based on existing columns in dataframe.
135    visualizer.add_line_indicator(column_name="SMA_20", graph_id=1)  # adding SMA_20 as line drawing on Candlestick chart
136    visualizer.add_line_indicator("RSI_2", 3)  # adding RSI_2 as line drawing to a new row
137    visualizer.add_bar_indicator("SMA_50", 4)  # adding SMA_50 as bar chart drawing to a new row
138    visualizer.add_marker_indicator("candle_bearish_engulfing_3", 1)  # adding markers based on candle_bearish_engulfing_3 on Candlestick chart
139    visualizer.add_marker_indicator("bearish_pinbar", 1) # adding custom indicator to Candlestick chart
140
141    # Step 8. Create figures ordered by a column from transaction file.
142    visualizer.create_figures(column="profit", n=10, ascending=False, window=14, show=True)
143
144    # Step 9. Save figures in html on local directory
145    visualizer.save_figures("temp_plotly.html")

Add indicator(s) based on stock price dataframe into the visualizer

  • define the indicators that you want to visualize based on indicators that are already in the stock price dataframe

  • 3 types of drawings are supported:

    • line chart drawing for number type indicator input: visualizer.add_line_indicator(column_name=”SMA_20”, graph_id=1)

    • bar chart drawing for number type indicator input: visualizer.add_bar_indicator(“SMA_50”, 4)

    • marker chart drawing for boolean type indicator input: visualizer.add_marker_indicator(“candle_bearish_engulfing_3”, 1)

  • user can display indicator on price chart, volume chart or new row of chart based on graph_id:

    • when graph_id = 1, the indicator will be displayed on Candlestick chart (eg, SMA_50, 1)

    • when graph_id = 2, the indicator will be displayed on volume chart

    • when graph_id > 2, the indicator will be displayed on a new row of chart (eg, RSI_2, 3)

1visualizer.add_line_indicator(column_name="SMA_20", graph_id=1)  # adding SMA_20 as line drawing on Candlestick chart
2visualizer.add_line_indicator("RSI_2", 3)  # adding RSI_2 as line drawing to a new row
3visualizer.add_bar_indicator("SMA_50", 4)  # adding SMA_50 as bar chart drawing to a new row
4visualizer.add_marker_indicator("candle_bearish_engulfing_3", 1)  # adding markers based on candle_bearish_engulfing_3 on Candlestick chart
5visualizer.add_marker_indicator("bearish_pinbar", 1) # adding custom indicator to Candlestick chart

Create figures ordered by a column from transaction file

  • generate figures based on 4 parameters:

    1. column: a column in transaction file that will be used for ordering completed transaction (eg, “profit” or “gain/loss”)

    2. n: number of completed transaction to display (eg, 10 = visualizing 10 figures)

    3. ascending: column’s value order by either True (from smallest to largest) or False (from largest to smallest)

    4. window: display number of bars before entry and after exit (eg, 14 = 14 bars before entry and 14 bars after exit)

    • [Optional] show: True (display the figures after being generated) or False (no display is needed)

    • [Optional] row_heights: the size of each chart (eg, [0.7,0.1,0.1,0.1] means 70% height for Candlestick chart, 10% height for volume chart, 10% for 3rd row chart, 10% for 4th row chart)

1#  Generate top 10 figures based on "profit" column in transaction file, and display it afterward
2visualizer.create_figures(column="profit", n=5, ascending=False, row_heights=[0.7,0.1,0.1,0.1], window=14, show=True)
